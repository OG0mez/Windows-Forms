In this solution contains 4 other projects added together as Windows Forms, along with the Login system itself. Submitted as a final project for class.

Uses a SQLConnection to Login to the database. From there, you can select a form to view out of:
  1. View different kinds of detailed PC Information.
  2. Live sync of two directories.
  3. FTP Connect will allow you to connect to your FTP server and make changes. 
  4. A sipmle enhanced Rich Text Editor form with all the main functions of an editor.
  
 There may be some small bugs but nothing major.
